# Template Examples

This folder contains example templates that you can copy to your `config/templates/` directory and customize.

## Quick Start

1. Copy the templates you want to customize:
   ```bash
   cp docs/examples/templates/*.j2 config/templates/
   cp docs/examples/templates/*.yaml config/templates/
   ```

2. Edit the templates in `config/templates/` to match your preferences

3. Restart JFC to apply changes

## Available Templates

### Jinja2 Templates (.j2)

| Template | Description |
|----------|-------------|
| `base_structure.j2` | Main poster layout structure (text positioning, scene format) |
| `visual_signature.j2` | Prompt for generating visual signatures from metadata |
| `scene_description.j2` | Prompt for generating scene descriptions |

### YAML Configuration (.yaml)

| File | Description |
|------|-------------|
| `category_styles.yaml` | Artistic styles per category (FILMS, SERIES, CARTOONS) |
| `collection_themes.yaml` | Color palettes and moods based on collection name keywords |

## Customization Tips

### Adding a New Category

Edit `category_styles.yaml`:

```yaml
DOCUMENTARIES:
  poster_style: "Documentary"
  base_mood: "Authentic, thought-provoking, journalistic"
  scene_context: "documentary exploration scene"
  lighting_style: "Natural lighting, candid photography style"
```

### Adding Collection Themes

Edit `collection_themes.yaml` to add themes based on keywords in your collection names:

```yaml
classics:
  color_hint: "sepia + gold + warm amber"
  mood_hint: "timeless, nostalgic, golden age"
  scene_hint: "classic film reel with vintage aesthetics"

anime:
  color_hint: "vibrant pink + electric blue + neon"
  mood_hint: "dynamic, energetic, Japanese animation style"
  scene_hint: "stylized action with speed lines"
```

### Visual Signatures

Visual signatures are generated by GPT from movie/series metadata and cached automatically in `data/cache/visual_signatures_cache.json`. You can customize the generation prompt by editing `visual_signature.j2`.

### Modifying the Poster Layout

Edit `base_structure.j2` to change:

- Text positioning (percentages from edges)
- Font styles and sizes
- Logo text and positioning
- Scene rendering instructions

**Warning**: Major changes to the layout may affect text readability. Test with a few posters first!

## Template Variables

### base_structure.j2

| Variable | Description |
|----------|-------------|
| `poster_style` | Style type (e.g., "Cinematic blockbuster") |
| `category` | Category name (FILMS, SERIES, CARTOONS) |
| `collection_display_name` | Collection name to display |
| `scene_description` | Generated scene description |
| `color_palette` | Color palette to use |
| `mood_style` | Mood and style description |
| `lighting_style` | Lighting instructions |

### visual_signature.j2

| Variable | Description |
|----------|-------------|
| `genres` | Genre names (comma-separated) |
| `overview` | Movie/series overview/synopsis |

### scene_description.j2

| Variable | Description |
|----------|-------------|
| `collection_name` | Name of the collection |
| `category` | Category type |
| `mood_hint` | Mood/atmosphere hint |
| `color_hint` | Color palette hint |
| `visual_signatures` | Visual signatures from collection items |

## Fallback Behavior

If a template file is missing from `config/templates/`, JFC will use the built-in defaults. This allows you to:

- Only customize the templates you need
- Update JFC without losing your customizations
- Reset a template by deleting it from `config/templates/`
