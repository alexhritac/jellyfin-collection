# Cookbooks - Jellyfin Collection (JFC)

Guides reutilisables pour les taches complexes.

## Creer un Cookbook

```
/agent-factory:create-cookbook
```

## Utilisation

Les cookbooks sont proposes automatiquement en Phase 0 de `/agent-factory:task`.

## Format

Chaque cookbook contient:
- Contexte et objectifs
- Etapes d'implementation
- Patterns et exemples de code
- Pieges a eviter
